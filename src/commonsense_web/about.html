About

We are a group of three undergrads from the University of Washington doing a project for the CSE 454 Advanced Internet Systems course.

In recent years, the field of Artificial Intelligence has expanded greatly, giving computers the ability to closely mimic human behavior and thought patterns with the exception of answering questions deemed as common sense.To solve this problem, an extensive data set has been compiled to include any and all information needed to answer these "common sense" questions. Our project focuses on creating a system that will answer a subset of these queries related to common sense size comparisons.
